stob

A lightweight STOB markup language C++ library. See more info at http://stobml.org

See WiKi for installation instructions.

STOB document example:

"String object"
AnotherStringObject
"String with children"{
    "child 1"
    Child2
    "child three"{
        SubChild1
        "Subchild two"

        Property1 {Value1}
        "Property two" {"Value 2"}
        //comment

        /* multi line
           comment */

        "Escape sequences \" \n \r \t \\ \/"
    }
}
