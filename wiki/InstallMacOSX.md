#Installation in MacOS X

Installation in MacOS X requires **[homebrew](http://brew.sh/)** package manager. Install it before proceeding.

- Add the homebrew tap (needs to be done once):

  ```
  brew tap igagis/tap
  ```

- Update formulaes:

  ```
  brew update
  ```

- Install **libpuu** keg

  ```
  brew install libpuu
  ```
